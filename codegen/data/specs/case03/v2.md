# balanced_parens

Return `True` if and only if every bracket in the string is correctly matched and nested.

- The bracket types are `()`, `[]`, and `{}`.
- **All other characters are ignored** (letters, digits, spaces, punctuation, etc.).
- A closer must match the most recently opened, still-unclosed opener of the same type.
- The empty string is balanced (`True`).

## Examples

- `balanced_parens('()')` → `True`
- `balanced_parens('a(b[c]d)e')` → `True`
- `balanced_parens(')(')` → `False`
- `balanced_parens('{[}]')` → `False`
